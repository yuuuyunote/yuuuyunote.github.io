Discord Server

日本製の Discord Bot をつなぐ場所です。

架け橋は、特定の思想や利害に偏らない中立な立場で運営されているコミュニティです。開発者には実利のあるサポートを、利用者には信頼できるBotとの出会いを提供します。

Discordに参加する 参加Botを見る
// WHY KAKEHASHI

他サービスとの違い

3つの方針によって、高品質なコミュニティをお届けします

01

中立性

サーバーには、主にDiscord標準機能で構成しています。また、運営メンバーもbotの開発とは無関係な人で構成しており、特定のbotに偏らない仕組みを作っています

02

利便性

参加者にとって実際に役立つ情報交換やサポートの集約を重視しています。形だけの交流にとどまらない、運営をおこなっています。

03

フラットな運営

サーバー主を含め、運営陣は上下関係のないフラットな体制で活動しています。特定の意見が優先されることのない、対等な話し合いを基本としています。

// HOW IT WORKS

サーバーの仕組み

運営方針を実際の設計に落とし込んだ、4つの取り組みです。

サポート充実

日本のbotが成長していく、システムを用意しています。

スマートなオンボーディング

公式機能による属性ロール付与とチャンネル表示の最適化を自動化しています。

スパム対策

自作Botにより、認証後の投稿制限を状況に応じて即時に解除する仕組みを備えています。

フォーラムの活用

話題ごとにフォーラムチャンネルを使い分け、チャンネル数の肥大化を防いでいます。

まずはサーバーをのぞいてみませんか。

参加は無料です。開発者・利用者どちらの立場でも歓迎しています。
